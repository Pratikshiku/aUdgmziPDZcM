### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 论坛
5. 公告
6. 流浪动物
7. 个人中心
8. 收货地址
9. 流浪动物收藏
10. 流浪动物近况
11. 流浪动物领养
12. 流浪动物助养


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 帖子类型管理
5. 公告类型管理
6. 动物类型管理
7. 消费类型管理
8. 流浪动物状态管理
9. 论坛管理
10. 公告管理
11. 流浪动物管理
12. 流浪动物留言管理
13. 流浪动物收藏管理
